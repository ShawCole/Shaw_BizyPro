### Chart Components (5 types, all in src/components/charts/)
- **HorizontalBarChart** — flex-fill container, min-h-[280px] → upgraded to flex-1 height="100%", pb-3
- **VerticalBarChart** — flex-fill, smart angle detection (label px vs slot width), dynamic yAxisWidth, dy={5} for label spacing
- **DoughnutChart** — flex-fill, full + semi-doughnut, legend, center percentage
- **TreemapChart** — flex-fill, intersection observer staggered fade-in (80ms per branch), isAnimationActive={false}
- **USAChoroplethMap** — cursor-following tooltip (tracks mouse position relative to container), quantile color scale

### Chart Fixes Applied
- Tooltips: solid #0F172A bg, white labelStyle, slate itemStyle, box-shadow, cursor={false} on bar charts
- Horizontal bars: removed expand/collapse ("Show All"), cleaned up unused state
- Vertical bars: smart angle (-30°) only when labels overflow slot width, XAxis height 18/40px, dy=5 gap
- All chart cards: flex flex-col + flex-1 min-h-0 wrapper + ResponsiveContainer height="100%" for grid-consistent sizing
- Choropleth: full-width col-span-full, always sorted last in rendering
- Treemap: flex-fill to use full card height
